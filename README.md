# JWT Authentication Flow - Chapter 06

**PRN232 - Implement Security in RESTful Web Service**

Giao diện presentation về JWT Authentication Flow theo phong cách ByteByteGo.

## Tính năng

- 📊 **Diagram ByteByteGo Style**: Sơ đồ luồng JWT Authentication với thiết kế clean, chuyên nghiệp
- 📑 **Slideshow**: Trình bày nội dung Chapter 06 qua các slide
- 🎨 **Dark Theme**: Giao diện tối phù hợp phong cách ByteByteGo

## Nội dung slides

1. Giới thiệu Chapter 06 - JWT Authentication Flow
2. JWT Authentication Flow Diagram (ByteByteGo style)
3. Cấu trúc JWT (Header, Payload, Signature)
4. Các bước trong quy trình xác thực
5. Best Practices bảo mật JWT
6. Kết thúc

## Chạy project

```bash
# Cài đặt dependencies
npm install

# Chạy development server
npm run dev
```
